# 📅 Day 2 – Git & GitHub

## What I Learned

- The difference between Git and GitHub.
- How to initialize a Git repository.
- How to stage files using `git add`.
- How to create commits.
- How Git keeps a history of my projects.
- How to debug Git problems using terminal commands.

## What I Built

- My first Git repository.
- My first and second Git commits.
- README file for my journey.

## Challenges

I accidentally created a Git repository inside my Windows user folder instead of inside my project folder.

I learned how to investigate the problem using:

- `git status`
- `git rev-parse --show-toplevel`

Then I fixed the repository correctly.

## Biggest Lesson

Don't panic when something breaks.

Debugging is part of programming.

Collect evidence first.
Fix second.

## Reflection

Today I felt like a real developer.

I solved my first Git problem instead of giving up.

That gave me confidence that I can learn much harder things in the future.
